The five-to-six-year handoff is already accounted for.

This is the main Linux PC now. When a newer system takes that role, this machine replaces the LOQ as the homelab and single-accelerator AI node.

Now: Linux gaming first

Install a current distro kernel, Mesa and linux-firmware so RDNA 4 support is not held back by old packages. Use the distro's upstream amdgpu/RADV stack for gaming.

Now: keep bulk storage external

The owned PiBOX dual-bay DAS handles SATA capacity over USB-C, so the compact airflow case stays uncluttered. Keep a separate backup regardless of RAID mode.

Later: grow memory, not motherboard

The four DIMM slots allow a future 64GB or 128GB configuration. PCIe 5.0 ×16 preserves the path to one faster accelerator without pretending this is a dual-GPU platform.

Keep lightweight services on the Vostro

Run SearXNG, Pi-hole and the MCP server on the 8th-gen i7 U-series laptop. It is the low-power control-plane box; the LOQ remains available for workloads that benefit from more CPU, RAM or the RTX 4060.

In five or six years: replace the LOQ

Move containers, build jobs and supported ROCm/inference experiments onto the 9700X/9070 XT desktop. The i7-13620H/RTX 4060 laptop can then retire from heavier always-on duty, while the Vostro continues hosting lightweight network services.
